## 📌 프로젝트 소개

"Who Is Him"은 퍼즐 형식의 웹 애플리케이션으로, 사용자가 단서를 통해 특정 인물을 추측하는 게임입니다.  
각 문제를 풀면서 그에 대한 정보를 점진적으로 알아가는 방식으로 설계되었습니다.

> **🤔 당신은 그를 알고 있나요? 퍼즐을 풀어보세요!**
